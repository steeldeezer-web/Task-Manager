# Task Manager Engine (Core Java Implementation)

## 📌 Описание
Консольный прототип системы управления задачами, созданный для отработки фундаментальных концепций Java Core и многопоточности. Проект демонстрирует эволюцию от хранения данных в памяти до интеграции с реляционной БД.

## 🚀 Ключевые особенности и технические решения:
* **Concurrency & Thread-Safety**: 
    - Реализована защита от Race Condition при одновременном создании задач.
    - Использование `ConcurrentHashMap` и `synchronized` блоков для обеспечения целостности данных.
    - Атомарные операции через метод `compute()` для минимизации блокировок.
* **Database Persistence**:
    - Интеграция с PostgreSQL через **JDBC**.
    - Реализация паттерна **DAO (Data Access Object)** для разделения бизнес-логики и SQL-запросов.
    - Работа с `PreparedStatement` для защиты от SQL-инъекций.
* **Architecture**: 
    - Четкое разделение на слои: Model, Service, Repository.
    - Использование `UUID` в качестве первичных ключей для обеспечения уникальности в распределенных системах.

## 🛠 Технологический стек:
* **Language**: Java 17+
* **Database**: PostgreSQL
* **Build System**: Maven
* **Testing**: JUnit 5
